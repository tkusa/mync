# mync

my netcat

## Installation
```
make
```

## Usage
```
mync 0.1.0
usage: mync [-Bhlnuvz] [-e executable] [-p port]
            [destination] [port]
options:
      -B            base64 encode/decode TODO
      -e executable execute program
      -h            print help message
      -l            listen mode
      -n            avoid name resolution
      -p port       custom port
      -U file       upload local file TODO
      -u            udp mode
      -v            verbose output
      -z            zero I/O

```
